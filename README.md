# CUMT-计算机本科生毕业论文-Skill

面向中国矿业大学计算机本科生毕业设计（论文）的 Codex skill。

这个仓库提供：

- `SKILL.md`：主技能说明
- `assets/cumt-thesis-template.doc`：学校论文模板资产
- `references/template-spec.md`：学校模板结构与格式摘要
- `references/project-first-workflow.md`：基于真实项目写论文的工作流

适用场景：

- 根据中国矿业大学毕业设计（论文）模板写作或排版
- 基于项目源码、开题报告、任务书、样文生成论文初稿
- 将论文初稿落版到学校模板中

仓库中的 skill 标识名保持为 `cumt-thesis-writing`，以保证 Codex 发现与触发稳定；展示名称使用 `CUMT-计算机本科生毕业论文-Skill`。
